Shortcode Kullanarak Sitenizdeki Diğer Konulardan Alıntı Yapmanıza Yarar.

== Description ==

Bu Eklenti İle Oluşan `[alixcan_alinti slug=""]` Shortkodu İle Bloğunuzda Bulunan Yazıların Özetlerini Yazmakta Olduğunuz Yazı İçerisinde Alıntı Şeklinde Gösterebilirsiniz.

Örneğin: Şurdaki Yazımda Diyerek Şurdaki Kelimesine Link Vermek Yerine `[alixcan_alinti slug="alinti-yazisi"]` Yazarak Konuyu Yazı İçine Ekliyebilirsiniz.


== Installation ==

1. Indirilen .zip Dosyasını Açın İçinden Çıkan
1. `alixcan_alinti` Klasörünü `/wp-content/plugins/` İçerisine Yükleyin.
1. Eklentiler Sayfasından Eklentiyi Aktif Edin.
1. Yazılarınızda `[alixcan_alinti slug=""]` Şeklinde Kullanmaya Başlıyabilirsiniz.


##Documentation

Kullanılıcak Shortcode: `[alixcan_alinti slug=""]`

Kullanımı: slug="" adlı alana daha alıntı yapmak istediğiniz yazının kalıcı bağlantısındaki `www.siteniz.net/genel/konu.html` `genel/konu` Bölümünü alarak `slug=""` Bölümüne Yapıştırıyorsunuz Ve Konuyu Alıntı Yapmış Oluyorsunuz.


Daha Fazla Yardıma İhtiyaç Duyarsanız [Tıklayın.](http://www.alixcan.net/?p=1608).
